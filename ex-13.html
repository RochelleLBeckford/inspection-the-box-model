# The Box Model


  • Everything on a web page is in a box. Every element on a page follows the box model:
  • The "box model" describes how an element handles its content and the space around it
  • Every element is made of four box layers in the following order, from the innermost to the outermost:
    • A "content box" for things like text and images
    • A "padding box" of space around the content
    • A "border box" that goes arounnd the padding
    • A "margin box" of empty space that surrounds the whole element
  • This is what the box model looks like in the browser:
  • The elements can take different shapes and sizes, but they all follow the box model.
